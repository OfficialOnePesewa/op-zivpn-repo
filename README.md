# 🚀 opudp - ZIVPN Pro Management Dashboard

**opudp** is a high-performance, automated management script designed for ZIVPN on VPS servers. Built with a focus on security, branding, and ease of use, it features **Strict Android Device Binding** and a comprehensive 19-option control menu.

---

## 💎 Exclusive Features
* **Strict Device Binding:** Enforce HWID (Device ID) security. No one can use a configuration file on a different device once bound.
* **VPS Intelligence:** Auto-detects OS (Debian/Ubuntu), Server Location, Country Flag, and System Specs.
* **User Management:** Add, remove, renew, and automatically clean up expired users via Cron Jobs.
* **System Tools:** Integrated Speed Test, Live Logs, and automated Bandwidth/Expiry reports.
* **Data Protection:** Professional Backup and Restore system for your user database.
* **Branding:** Fully branded under the **opudp** name.

---

## 🛠 Supported Operating Systems
The script is optimized for the following (x86_64 & ARM):
* **Ubuntu:** 20.04, 22.04, 24.04+
* **Debian:** 10, 11, 12+
